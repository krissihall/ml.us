Brunner Studio

Responsive Website

My Involvement: JavaScript/jQuery, Back-End Development
Project Manager: Alane Wholey
Designer: Flav Rosati
Front-End: Steven Boothe, Chris Jezowicz
Back-End: Steven Boothe, Sabrina Shaner

This is the brand-new Brunner Studio website that is built to showcase the work that the Digital Studio within Brunner handles. It is a responsive site with heavy use of JavaScript and jQuery for the slideshows and the top navigation. It is also built to be easily updated via a asp.NET Content Management System.

My part of this project was setting up the Content Management System for maintainence on this site. I also helped to fine-tune the media queries and cascading style sheets as well has helping to debug the JavaScript that makes the slideshows function. This was a little bit tricky as the design was revised at a few different stages in development to make the pieces work better as a whole.
