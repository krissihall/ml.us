Wise Snacks

Website

My Involvement: Front-End Development, Back-End Development, Facebook Integration
Project Manager: Erin Dolhi
Designer: Swara Rane

This was one of the first asp.NET projects I worked on and I knew only the basics going into it as we had primarily learned and used PHP during college and my internship. This website was built using HTML, CSS, and JavaScript/jQuery and was built to be maintained via an asp.NET Content Management System. We no longer perform maintainence on this website.

This site was built prior to the Responsive boom so is best viewed on a desktop computer. There is quite a bit of Facebook integration with Recipe Share buttons that will post to the users Facebook Wall. There is also an individual like button for each product that is created dynamically along with the publishing of a product. The contact forms were all handled by a back-end developer and content population was handled by the Project Manager and an Intern.
