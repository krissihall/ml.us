Atria's Oktoberfest

Responsive Micro-Site

My Involvement: Front-End Development, HTML5 Animation, JavaScript/jQuery, Reponsive Break-Down for Mobile Sizes
Designer: Flav Rosati

This micro-site was a one page promotional piece that was announcing some coupon opportunities for Atria's Restaurant during their Oktoberfest event in Fall 2012. The "steins" rotate on a custom-coded carousel and reveal the text in English by clicking on the 'translate' button below each "stein". This piece was built to be responsive and every piece of it was hand-coded rather than relying on JavaScript or jQuery plugins.

One of the tricky things about this piece was to make the positioning change as the browser was sized down. I also decided that for the mobile sizes, only one "stein" should show since otherwise, you would not be able to read the translated text as they would all be too small.
