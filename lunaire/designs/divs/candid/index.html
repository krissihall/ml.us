Rather pretty if I say so myself. I really enjoyed making this one. I also am into really simplified css and not too much by way of graphics. I wonder if that means what I think it does. Well it's another ayumi layout, hopefully not too difficult to use.

This layout was made in photoshop cs2, using brushes and textures from Hybrid Genesis and Aunexamor.net. It didn't take too long to design and code (about 2.5 hours).

Usage

The only file you need to edit in this zip is index.html. In index.html, only the parts marked "Content" and "Navbar" should be edited, and nothing else. Please be aware that it relies heavily on cascading style sheets (css) so you follow the basic setup of this page and use it as the template for your other pages. Also don't forget to upload the stylesheet or your layout won't work properly.
