Since people liked my patchwork layout I figured that I'd try out the style again, this time using an anime image. Not exactly kid friendly but then again, I don't design for kids. You can use this for a blog preferably, but it can be used for a personal website as long as it's not a large one. Please note that this layout uses an external style sheet. Ensure that your host supports .css files because some don't (o_O yeah, amazingly). Also, it has been tested in IE 6 and Mozilla and it looks fine in them. I went through the trouble of making sure that this document contains both valid html and css and I proud to say that it does :)

Usage

Using WinZip, WinRar or whatever other program that you use to unzip your files, unzip all your files to one folder. Then upload everything to the same directory. Remember to keep all folder paths which means that you need to create a sub directory or folder called images and then upload all the images there. If not then you will find broken images, non formatted pages ect.

You must have some basic html knowledge and CSS knowledge as this uses heavy CSS. DO NOT edit the demure.css file unless you know what you're doing. Edit only the specified areas in the index file but please leave the rest alone. Note that when uploading this layout, unless you changed the html file name, it will replace your original index file so make sure you want that to happen when uploading this.
