About The Layout

This design is so pretty, I had fun designing it. I completely adore the main image and especially the images from this series. The magna carta oxide collection is rather lovely if you ask me. For this layout, yellow was a natural choice. I'm just in love with the colors of the layout. It's not too overpoweringly yellow, and not blindingly bright either. Some people may not like the whole pixel stretch thing but I think it looks right at home here.

This layout uses brushes, pattens and textures, all of which I made myself. The layout itself has a heavy use of divs and css coding. This layout is not recommended for html newbies.

Using This Layout
The only file you need to edit in this is index.html. In index.html, only the parts marked between the content and navbar areas should be edited, and nothing else.  All credits must be kept in their original positions, unchanged. Please be aware that it relies heavily on cascading style sheets (css) so you follow the basic setup of this page and use it as the template for your other pages when implimenting it through your whole website. Also don't forget to upload the stylesheet or your layout won't work properly.
