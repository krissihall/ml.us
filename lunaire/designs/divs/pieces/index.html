A very simple layout which I intended to be for a fan site. Well I'm doing a different fan site so I decided to put this up for download.  This pretty but somber layout features the Japanese Singer Ayumi Hamasaki. I love the song and the video is touching.

You can use this for a blog preferably, but it can be used for a personal website as long as it's not a large one. Please note that this layout uses an external style sheet. Ensure that your host supports .css files because some don't (o_O yeah, amazingly). Also, it has been tested in IE 6 and Mozilla and it looks fine in them. I went through the trouble of making sure that this document contains both valid html and css and I proud to say that it does :)

Usage

Using WinZip, WinRar or whatever other program that you use to unzip your files, unzip all your files to one folder. Then upload everything to the same directory. Remember to keep all folder paths which means that you need to create a sub directory or folder called images and then upload all the images there. If not then you will find broken images, non formatted pages ect.

You must have some basic html knowledge and CSS knowledge as this uses heavy CSS. DO NOT edit the pieces.css file unless you know what you're doing. Edit only the specified areas in the index file but please leave the rest alone. Note that when uploading this layout, unless you changed the html file name, it will replace your original index file so make sure you want that to happen when uploading this.
