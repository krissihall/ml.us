Orange. I don't think I've made an orange layout before. Scratch that. It's -Coral-. I wanted a new theme for my domain so I started making this for it, but then I got thinking that I need more downloadable layouts for my site so I'm making this one available for download.

I really like the headers on this one. They just make the layout look more lively and add more color to it. This layout was made in Photoshop cs2, using default tools with patterns and brushes that I made myself. The icon border brush cam from Aethereality and the little scribble heart I made myself. The main image is a silhouette from the anime Ai Yori Aioshi

Usage

The only file you need to edit in this zip is index.html. In index.html, only the parts marked between the content and navbar areas should be edited, and nothing else.  All credits must be kept in their original positions. Please be aware that it relies heavily on cascading style sheets (css) so you follow the basic setup of this page and use it as the template for your other pages. Also don't forget to upload the stylesheet or your layout won't work properly.
